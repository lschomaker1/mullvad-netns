mullvad-netns

Run selected apps/commands through Mullvad WireGuard in a Linux network namespace without changing host default routes.

Install

curl -fsSL https://mullvad-netns.schoma.kr/install.sh | \
  REPO_RAW_BASE="https://mullvad-netns.schoma.kr" bash
